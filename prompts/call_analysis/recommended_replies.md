# 추천 멘트 생성 프롬프트

다음 통화 분석 결과를 바탕으로 상담사가 사용할 추천 멘트를 생성하세요.

## 통화 내용
{{conversation}}

## 분석 결과
- **고객 상태**: {{customer_state}}
- **주요 니즈**: {{primary_need}}
- **고객 고민점**: {{pain_points}}
- **긴급도**: {{urgency_level}}

## 업종/회사 정보
{{script_context}}

## 생성 가이드라인

### 1. 상황에 맞는 멘트
- 고객 상태와 니즈를 반영
- 고객의 고민을 해결하는 방향

### 2. 톤 & 스타일
- 업종/회사 스크립트의 톤을 따름
- 자연스러운 한국어 존댓말
- 강압적이지 않은 부드러운 권유

### 3. 실용성
- 바로 사용할 수 있는 완성된 문장
- 3-5개의 다양한 상황별 멘트

### 고객 상태별 접근

**관심 있음**: 구체적인 정보 제공, 다음 단계 안내
**고민 중**: 추가 정보 제공, 비교 자료 안내
**망설임**: 우려 해소, 신뢰 구축 멘트
**구매 준비됨**: 클로징 멘트, 혜택 강조
**관심 없음**: 부드러운 마무리, 연락처 남기기

## 출력 형식

```json
{
  "next_action": "다음에 취해야 할 행동 (예: 견적서 발송 후 3일 내 후속 전화)",
  "recommended_replies": [
    "추천 멘트 1 - 현재 상황에 맞는 즉시 사용 멘트",
    "추천 멘트 2 - 고객 고민 해소 멘트",
    "추천 멘트 3 - 다음 단계 유도 멘트",
    "추천 멘트 4 - 클로징/마무리 멘트",
    "추천 멘트 5 - 후속 연락용 멘트"
  ],
  "reply_context": [
    {
      "situation": "가격 문의 시",
      "reply": "추천 멘트"
    },
    {
      "situation": "고민 표현 시",
      "reply": "추천 멘트"
    }
  ]
}
```

**중요**:
- 업종/회사 스크립트가 있으면 해당 스타일을 최대한 반영하세요
- 한국어 뉘앙스를 자연스럽게 유지하세요
- 실제 상담사가 바로 말할 수 있는 완성된 문장을 만드세요

JSON만 출력하세요.
