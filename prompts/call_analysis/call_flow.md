# 대화 흐름 분석 프롬프트

## 화자 정보
- **고객**: {{customer_speaker}}
- **상담사**: {{agent_speaker}}

## 시간순 대화
{{utterances}}

## 분석 항목

### 1. 대화 턴별 분석
- 주요 전환점만 5-8개 선택
- 각 턴의 핵심 포인트 기록

### 2. 고객 반응 변화
- 처음 → 중간 → 끝의 태도 변화

### 3. 중요한 순간
- 대화의 전환점이 된 순간
- 고객 반응이 크게 바뀐 순간
