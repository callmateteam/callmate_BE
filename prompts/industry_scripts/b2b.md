# B2B 영업 스크립트 가이드

## 업종 특성
B2B 영업은 의사결정자가 여럿이고, 논리적 설득이 중요합니다. ROI와 문제 해결에 집중합니다.

## 상황별 권장 멘트

### 1. 첫 인사 / 라포 형성
- "안녕하세요, [회사명] [이름]입니다. 담당자분 연결 부탁드립니다."
- "[담당자명]님, 좋은 시간에 연락드렸을까요?"
- "잠깐 통화 가능하시면 [서비스/제품] 관련해서 말씀드리고 싶어요."
- "바쁘시면 이메일로 자료 먼저 보내드릴까요?"

### 2. 니즈 파악
- "현재 [영역]은 어떻게 관리하고 계세요?"
- "기존에 사용하시는 솔루션이 있으신가요?"
- "가장 불편하신 점이 뭐예요?"
- "이 부분에서 가장 중요하게 생각하시는 게 뭔가요?"
- "예산은 어느 정도 잡고 계세요?"
- "의사결정 프로세스가 어떻게 되나요?"

### 3. 솔루션 제안
- "저희 [제품/서비스]가 딱 그 문제를 해결해요."
- "비슷한 규모의 [업종] 기업들이 많이 사용하고 계세요."
- "[고객사명]도 저희 고객인데, [성과]를 달성했어요."
- "데모 한번 보여드릴까요? 15분이면 충분해요."

### 4. ROI / 가치 제안
- "도입하시면 월 [시간/비용] 절감 가능해요."
- "투자 대비 [N]개월 안에 회수 가능합니다."
- "업무 효율이 [N]% 개선된 사례가 있어요."
- "인건비로 환산하면 [금액] 절약이에요."

### 5. 반대 처리 (이의 제기 대응)

#### "예산이 없어요"
- "올해 예산은 어려우시면, 다음 분기 예산에 반영하시면 어떨까요?"
- "파일럿으로 소규모 시작도 가능해요."
- "ROI 분석 자료 보여드릴까요? 비용 대비 효과 보시면 고민되실 거예요."

#### "결정권자가 따로 있어요"
- "그분께 직접 설명드리면 더 좋을 것 같은데, 미팅 잡아주실 수 있나요?"
- "자료 준비해드릴게요. 내부 공유하실 때 쓰시면 돼요."
- "의사결정에 필요한 정보가 뭔지 알려주시면 맞춰서 준비할게요."

#### "기존 시스템이 있어요"
- "기존 시스템과 연동도 가능해요."
- "비교 분석 자료 보여드릴까요?"
- "전환 비용 대비 장기적으로 훨씬 이득이에요."

#### "나중에 연락주세요"
- "언제쯤이 좋으실까요? 일정 잡아드릴게요."
- "그 전에 자료만 먼저 보내드릴까요?"
- "다음 분기에 검토 예정이시면, 그때 맞춰서 다시 연락드릴게요."

#### "비슷한 거 많잖아요"
- "맞아요, 근데 저희가 다른 점은 [차별점]이에요."
- "직접 비교해보시면 차이 느끼실 거예요."
- "특히 [기능/서비스]는 저희만 가능해요."

### 6. 클로징 (마무리)
- "그럼 다음 주에 데모 미팅 잡을까요?"
- "파일럿 진행해보시겠어요? 무료로 2주 가능해요."
- "계약서 초안 보내드릴까요?"
- "결정권자분과 미팅 일정 잡아주실 수 있나요?"

### 7. 후속 연락
- "보내드린 자료 검토해보셨나요?"
- "데모 후에 추가 질문 있으셨어요?"
- "내부 논의는 어떻게 되고 있나요?"
- "다음 단계로 넘어가려면 뭐가 필요하세요?"

## 주의사항
- 담당자 시간 존중
- 과장된 성과 언급 금지
- 경쟁사 비방 금지
- 의사결정 프로세스 파악 필수
