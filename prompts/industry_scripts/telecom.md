# 통신사 영업 스크립트 가이드

## 업종 특성
통신 영업은 요금제 비교와 혜택 설명이 핵심입니다. 가성비와 편의성을 강조합니다.

## 상황별 권장 멘트

### 1. 첫 인사 / 라포 형성
- "안녕하세요, [통신사명] [이름]입니다."
- "요금제 관련해서 문의 주셨죠?"
- "현재 사용하시는 요금제가 어떻게 되세요?"

### 2. 니즈 파악
- "한 달에 데이터 얼마나 쓰세요?"
- "통화는 많이 하시는 편이에요?"
- "현재 요금이 어느 정도 나오세요?"
- "가족결합이나 인터넷 결합은 하고 계세요?"
- "스마트폰 바꾸실 계획 있으세요?"

### 3. 요금제 제안
- "고객님 사용 패턴이면 [요금제명]이 딱이에요."
- "지금보다 [금액]원 정도 절약되실 거예요."
- "데이터 무제한에 통화도 무제한이에요."
- "이번 달 프로모션으로 [혜택]도 있어요."

### 4. 혜택 설명
- "가족결합하시면 추가 [금액]원 할인돼요."
- "멤버십 혜택으로 [혜택] 받으실 수 있어요."
- "인터넷 같이 쓰시면 결합 할인 크게 나와요."
- "최신 기기 할부 지원도 가능해요."

### 5. 반대 처리 (이의 제기 대응)

#### "지금 요금제도 괜찮아요"
- "혹시 현재 요금 얼마 나오세요? 비교해드릴게요."
- "같은 조건인데 더 저렴한 게 나왔어요."
- "데이터 더 많이 드리면서 가격은 그대로예요."

#### "약정 남았어요"
- "위약금 계산해드릴까요? 바꿔도 이득일 수 있어요."
- "약정 끝나면 연락드릴까요? 메모해둘게요."
- "일단 정보만 드릴게요. 약정 끝나면 참고하세요."

#### "다른 통신사도 알아볼게요"
- "비교해보시는 거 좋아요. 저희 강점 말씀드리면..."
- "어떤 부분 비교하시는지 알려주시면 맞춤 정보 드릴게요."
- "다 알아보시고 저한테 연락주세요. 최저가 맞춰드릴게요."

#### "통화품질 걱정돼요"
- "저희 커버리지 전국 최고 수준이에요."
- "불만족하시면 위약금 없이 해지 가능해요."
- "고객님 지역은 품질 문제없어요. 확인해봤어요."

### 6. 클로징 (마무리)
- "그럼 이 요금제로 변경해드릴까요?"
- "신분증만 있으시면 바로 처리 가능해요."
- "오늘 가입하시면 [혜택] 추가로 드려요."
- "방문 가입 원하시면 시간 잡아드릴까요?"

### 7. 후속 연락
- "변경하신 요금제 잘 쓰고 계세요?"
- "추가 문의사항 있으시면 편하게 연락주세요."
- "가족분들도 변경 원하시면 같이 해드릴게요."

## 주의사항
- 허위 요금 정보 금지
- 숨겨진 비용 명확히 고지
- 약정 조건 정확히 설명
- 과도한 권유 금지
